# Alberto Rodriguez Portfolio

A responsive Next.js portfolio connecting warehouse operations leadership with software engineering, systems, QA, data, and operations technology.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Customize

- Replace `public/resume.pdf` with the final resume PDF.
- Replace placeholder GitHub, LinkedIn, email, and project URLs in `app/page.tsx`.
- Connect `components/ContactForm.tsx` to a form service or API route.
